03
Buttons
Press down like a cartridge. Only for key actions.
B1 — Primary (Daily)
Blue. Daily game CTAs. Try hovering — a light chevron sweeps across. Try clicking — the 5px shadow collapses on press.
💡 Two interactions layered: (1) chevron sweep ::before pseudo-element, skewX(-20deg) sliding left→right on hover via chevronSweep keyframe. (2) box-shadow bottom-only: 5px at rest → 1px on :active + translateY(4px). Both work together — the sweep on approach, the press on commit.
B2 — Primary (Blitz) · Stripe
Amber, animated diagonal stripe texture at rest — kinetic even before you touch it. Blitz only. Makes it feel like something's already happening.
💡 Stripe uses linear-gradient (not repeating-) inside a background-size tile — the 25%/50%/75% colour stops create clean parallel stripes that tile seamlessly. Animation moves background-position by exactly one tile (28px 28px) → zero jumpiness. ::before pseudo covers the button, overflow:hidden clips to pill shape.
B3 — Dark / Confirm
Ink. Post to leaderboard, confirm. Same press.
B4 — Ghost
No press effect — ghost buttons are secondary. Never the main action.
B5 — Text Link (Blue)
Underline with offset 3px. Reopening modals, navigating to detail.
B6 — Text Link (Amber)
Blitz contexts, on dark backgrounds.

Stamp earn animation — the most important reward moment
+600 pts
Badge
The stamp-down
scale(0.4) from above → scale(1.18) overshoot → slight wobble → settles at scale(1.0). Two ink rings pulse outward. Score float rises and fades. This is the one animation moment that defines the whole reward system. It must feel physical — like an actual ink stamp hitting paper. Every frame matters.
💡 The wobble is crucial — scale(1.18) → scale(0.94) rotate(-2deg) → scale(1.05) rotate(1deg) → scale(1). Without it the stamp feels digital. With it, it feels physical. Game Sense blank reveal uses the same principle: rotateY(90deg) → rotateY(-4deg) → rotateY(0) — always overshoot then settle.

09
Micro-Interactions
Small design spells. Each one a surprise.
Button press
5px shadow collapses on :active. translateY(4px). Feels like pressing a key, not tapping a screen.
CSS only. Works on B1, B2, B3. Not B4/B5/B6 — they're secondary and shouldn't have the same weight.
Correct answer float
+1 correct ✓
Float rises from input, fades over 0.8s. HUD score bounces scale(1.35). Correct pill shoots into arena.
scorePop keyframe: translateY(0) → translateY(-44px) + opacity 1→0. HUD scoreBounce: scale(1)→scale(1.35)→scale(1) in 0.25s spring.
Wrong answer shake
Not this one
Input shakes horizontally. Red border flash 0.4s. Wrong pill appears in arena, fades after 0.8s.
shake keyframe already in tailwind.config. Border: transition to --red over 0.15s, return after 0.4s via JS class toggle.
Blank reveal (Game Sense)
A game about ___
The rotateY flip with overshoot. Card flips in from 90deg, overshoots to -4deg, wobbles to 0. Feels like turning over a flashcard.
rotateY(90deg) → rotateY(-4deg) scale(1.02) → rotateY(2deg) → rotateY(0). transform-style: preserve-3d on parent. 0.45s cubic-bezier(0.34,1.5,0.64,1). This is the Idle Hours signature animation — all reveals should use this principle.
Milestone unlock toast
🥈 Silver unlocked!
Slides down from above with spring bounce. Stays 2.2s, slides back. Milestone dot pulses amber glow simultaneously.
translateY(-80px) → translateY(4px) → translateY(0) with cubic-bezier(0.34,1.5,0.64,1). Not a simple slide — the overshoot is what gives it life.

10
Rules of Thumb
Paste these into Claude Code when things go wrong.
For Claude Code prompts
Familiar to use.
Surprising to look at.
1
Cards are white. Pages are cream.
Card = #FDFAF5 + real box-shadow. Cream card on cream page = invisible. Internal dividers are dashed, not solid.
2
Scores are T2 — 68px, 900 weight.
The result number is the headline. Minimum 48px. --ink. Never blue. Never in a pill.
3
Stat pills are full pill. Chips are labels.
Stat pills: border-radius 100px, no border, --cream-dark fill. Context chips: border-radius 6px, with border. These are different shapes for different jobs.
4
Badges have inner rings.
Every filled badge circle: 7px inset inner ring at rgba(255,255,255,0.22). Matches brand PDF illustrations. Non-negotiable.
5
Primary buttons press down.
box-shadow: 0 5px 0 [darker] at rest. Collapses to 1px on :active + translateY(4px). B1, B2, B3 only. Not ghost or text links.
6
Animations overshoot and settle.
Every reveal, stamp, toast, and card hover uses cubic-bezier(0.34,1.5,0.64,1). The overshoot is what makes it feel physical. Never ease-in-out.
7
Card hover direction is intentional.
Topic rows: translateX (shelf). Game cards: translateY + rotate(-0.5deg) (picking up). Your row in leaderboard: amber left border + amber name. Never just translateY(-2px) everywhere.
8
Icons are Idle Hours set, not emoji.
Custom SVG outlines, rounded ends, consistent stroke weight. Use placeholder boxes until drawn — don't substitute emoji. Emoji in UI = breaking the language.
9
Stripe texture belongs on active/reward elements.
B2 button always. Milestone toasts always. Score chips on end screens. HUD score container during Blitz. Never on body text, never on cards. It signals energy — dilute it and it means nothing.
10
Stickers rotate. Never perfectly straight.
Alternate -1.5deg and +1.5deg across a group of stickers. One per card maximum. The slight tilt is what makes them feel placed by a human, not rendered by a machine.
