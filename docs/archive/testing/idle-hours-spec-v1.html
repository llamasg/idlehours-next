Inertia Tooltip (cursor-following)
Move your cursor across the track — the tooltip lags behind with a spring tail.
Hover me
Implementation: mousemove updates a CSS variable, tooltip left transitions with 0.25s var(--ease-out) — the lag is the transition delay between cursor position and tooltip position. No JS physics required. Use for score trackers, slider handles, any draggable UI.

The rules
Making It Feel Made
Everything tilts. Nothing is straight. Stickers always at -1.5 or +1.5 degrees.

The cream is never pure white. It breathes.

Labels are labels. They do not become buttons.

Stripe texture is energy. Use it once per screen, for Blitz.
The stipple background is the desk surface. Sticky notes live on it.

Shadows go down and slightly right. Never inward.

Rounded corners are never the same radius twice — cards, chips, pills each have their own.

Motion has weight. Nothing slides. Things land.

Section 10
Implementation Rules
Non-negotiables. These are the decisions already made.
1
Never transform flex/grid containers
GSAP and CSS transforms applied to a flex or grid container break child layout. Always wrap the content in an inner div and transform that instead.
✓ DO — animate the inner wrapper div
✗ DON'T — gsap.to('.grid-container', { y: 20 })
2
Set opacity:0 inline before first paint
Any element that starts hidden must have style="opacity:0" in the HTML. CSS classes applied after mount may flash.
✓ DO — inline style opacity:0, then GSAP sets to 1
✗ DON'T — rely on className to hide on mount
3
One accent colour per screen
Each game world uses one accent. On hub pages (outside a game), blue is Game Sense, amber is Blitz/editorial warmth. Never render two game accents simultaneously.
✓ DO — set accent via gameTheme prop, scope to layout
✗ DON'T — blue CTA next to purple card next to amber tag
4
Shape grammar is fixed
Stat pills = border-radius:100px. Context chips = 6px. Cards = 16px. Major cards = 20px. Popup = 22px. Do not mix radii within the same component family.
✓ DO — pill for stat, chip for tag, card for content
✗ DON'T — 12px on a pill, 100px on a content card
5
Stripe texture = Blitz and energy moments only
The animated diagonal stripe background is reserved for Blitz (always on) and high-energy moments like score milestones. Never used decoratively on cards, headers, or backgrounds.
✓ DO — Blitz button, Blitz game world, milestone popup
✗ DON'T — hero banner, card decorations, nav bar
6
Reduced motion — skip everything
Check prefers-reduced-motion on mount. If true: skip all GSAP timelines, set all opacities to 1 immediately, disable circle wipe (use instant background swap instead), disable streak animation.
✓ DO — useReducedMotion hook, skip GSAP entirely
✗ DON'T — halve durations, still run animation loops
